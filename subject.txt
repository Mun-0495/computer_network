### **1. 과제명**
- AI를 활용한 채팅 프로그램 설계와 구현

 

### 2. 과제의 목적
- 본 과제는 정답이 없습니다. AI와 협업하여, 얼마나 사용자 관점에서 생각하고 설계했는가를 중요하게 보고자 합니다.

 

### 3. 과제 내용
1) 첨부된 multi_sever.cc 프로그램과 통신하는 클라이언트 프로그램 구현

     - 기본 기능 구현 : 서버와 정상 연결, 송수신, 종료 

     - 확장 기능 구현  : 자신만의 기능 추가 (예시 - 닉네임, 접속 및 퇴장 알림, 귓속말, 접속 시간, 텍스트 색상, 이모티콘 등등)

 

2) multi_server.cc 업데이트 (클라이언트에 기능을 추가할 때 마다 서버 코드 업데이트 불가피)

     - multi thread 방식은 유지 (single thread 형태로 변경하면 안됩니다.)

 

3) 구현 내용과 설계에 관련한 내용은 소스 파일 내에 주석으로 작성
